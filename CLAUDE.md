서비스 개요
- 광고 제안을 종이로 한 거를 홈페이지로 제작하여 URL로 제공을 할거임

서비스 특징
- 전체적인 레이아웃 구성은 개발자가 직정 할 예정이며 상세 내용은 노션 API를 사용하여 노션 블록들을 불러와서 작업 예정

프로젝트 구성
- REACT.JS 최신 버전
- NEXT.JS 최신 버전
- TypeScript 최신 버전
- ShadCN UI 최신 버전
- NOTION API (fetch option 사용)
